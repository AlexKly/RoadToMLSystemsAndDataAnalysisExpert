# Список литературы по специализации "Математика и Python для анализа данных"
<p> Мы составили для вас список наших любимых учебников по темам, рассматривавшимся в этом курсе, с короткими комментариями. </p>
<h3> Линейная алгебра </h3>
<ul>
	<li> Ильин, Ким. <i>Линейная алгебра и аналитическая геометрия</i> (1998) — МГУ. </li>
	<li> Умнов. <i>Аналитическая геометрия и линейная алгебра</i> (2011) — МФТИ. </li>
	<li> Деммель. <i>Вычислительная линейная алгебра. Теория и приложения</i> (2001) — понятный кусок про матричные разложения. </li>
</ul>
<h3> Математический анализ </h3>
<ul>
	<li> Ильин, Позняк, <i>Основы математического анализа</i> (2005) — МГУ. </li>
	<li> Тер-Крикоров, Шабунин. <i>Курс математического анализа</i> (2001) — МФТИ, много примеров. </li>
	<li> Иванов. <i>Лекции по математическому анализу</i> (2000) — МФТИ, очень короткое, но полное изложение. </li>
</ul>
<h3> Методы оптимизации </h3>
<ul>
	<li> Нестеров. <i>Методы выпуклой оптимизации</i> (2010) — математически строгое введение в оптимизацию от живого классика. </li>
	<li> Boyd, Vandenberghe. <i>Convex Optimization</i> (2004) — идеальная книга по классической оптимизации, много интересных постановок задач. </li>
	<li> Schneider, Kirkpatrick. <i>Stochastic Optimization</i> (2006) — стохастическая оптимизация во всём многообразии. </li>
</ul>
<h3> Теория вероятностей и статистика </h3>
<ul>
	<li> Dekking, Kraaikamp, Lopuhaa, Meester. <i>A Modern Introduction to Probability and Statistics, Understanding Why and How</i> (2005) — доступная книга, описывающая базовые понятия, теоремы и методы; разбирается очень много примеров, тесно связанных с задачами машинного обучения и анализа данных. </li>
	<li> Лагутин. <i>Наглядная математическая статистика</i> (2007) — в основном статистика, но есть и небольшое введение в теорию вероятностей. Стоит читать, кроме глав про классификацию и анализ данных, там изложение не слишком современно. </li>
	<li> Чжун, АитСахлиа. <i>Элементарный курс теории вероятностей. Стохастические процессы и финансовая математика</i> (2007) — очень простое изложение. </li>
	<li>
		<p> Отличные лекции с мехмата Новосибирского Государственного Университета: </p>
		<p> http://www.nsu.ru/mmf/tvims/chernova/tv/tv_nsu07.pdf — теория вероятностей, </p>
		<p> http://www.nsu.ru/mmf/tvims/chernova/ms/ms_nsu07.pdf — математическая статистика. </p>
	</li>
	<li> Diez, Barr, Cetinkaya-Rundel, Dorazio. <i>Advanced High School Statistics</i> (2015) — вводная книга, программа соответствует типичному курсу Statistics 101 хорошего западного университета. </li>
	<li> DasGupta. <i>Probability for Statistics and Machine Learning: Fundamentals and Advanced Topics</i> (2011) — для смелого читателя, рассматриваются в том числе достаточно высокоуровневые методы. </li>
</ul>
<h3> Python </h3>
<ul>
	<li> Классические руководства по Python: https://docs.python.org/2/tutorial/ (2.7), https://docs.python.org/3/tutorial/ (3.5) </li>
	<li> Reitz. <i>The Hitchhiker’s Guide to Python</i> http://docs.python-guide.org/en/latest/ — довольно полное руководство, в котором рассматриваются вопросы от установки, работы с виртуальным окружением и работы в различных IDE до основных структур языка с примерами кода. </li>
	<li> <i>Google python class</i> https://developers.google.com/edu/python/ — небольшой бесплатный онлайн-курс по Python для слушателей с минимальным опытом программирования. </li>
	<li> Lutz. <i>Learning Python</i> (2013) — с этой книги можно начинать изучение, она покрывает все основные структуры языка. </li>
	<li> Lutz. <i>Python Pocket Reference</i> (2015) — подробный справочник. </li>	
</ul>
